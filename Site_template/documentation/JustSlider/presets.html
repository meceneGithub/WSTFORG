To create presets please use this model: :









Preset:

Attributes to edit:

  • reverse way – reverse the direction of animation;
  • duration – the duration of one animation block;
  • interval – the period of time in which one animation is followed by another;
  • blocksX – the number of blocks located horizontally;
  • blocksY – the number of blocks located vertically;
  • easing - Easing pattern;
  • way - the way animatuion runs;
  • animation - the animation type;

To create the new preset, please use the ready-made one, change certain settings and then click get preset. As soon as it is clicked you will see the line that you should copy into the file tms_presets.js. You should add it into the object presets:{} - located at the top of the file, in the same way the other presets are listed.

It is important to set the unique name that will be used for the preset during the initialization:

uniquename:{"reverseWay":false,"duration":"1000","interval":"1","blocksX":"1","blocksY":"1","easing":"","way":"lines","anim":"fadeThree"}
